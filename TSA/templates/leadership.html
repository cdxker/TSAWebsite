LEADERSHIP

2019-2020 Dubiski Chapter Officer Application


Interested In Being A State Officer? To apply to run for a Texas TSA state officer position you will need to fill out this application. All applicants must be in current grade 7 – 11.
How To Apply for State Office

Application is due by February 1.

The following items are needed to have a complete application:
  • Completed Application
  • Completed Advisor’s Recommendation
  • Completed Administrator’s Recommendation
  • Completed Community Member’s Recommendation
  • Candidate Photo
If you have any questions about the officer application process please contact Jennifer O’Brian at: officer.coordinator@texastsa.org

State Leadership Team Application:

The state leadership team is made up of six underclassman. The role of the leadership team member is to aide the state officers. Leadership team members must be in current grades 6-10. To apply for the state leadership team please fill out this application. Applicants that are selected to serve on the leadership team will be informed of their selection at state contest.
How to Apply for State Leadership Team
Application is due by February 1.
The following items are needed to have a complete application:
  • Completed Application
  • Completed Advisor’s Recommendation
  • If you have any questions about the officer application process please contact Jennifer O’Brian at: officer.coordinator@texastsa.org
